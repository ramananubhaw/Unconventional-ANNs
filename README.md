<h1><b>Unconventional ANNs</b></h1>

<p>Artificial Neural Networks redesigned with a <b>non-standard</b> matrix structure.</p>
<p>Each <b>row</b> in the matrix represents a <b>neuron</b>, not a batch sample as in conventional ANNs.</p>
<p>All core components - <b>Dense Layer</b>, <b>Activation Functions</b>, <b>Loss Functions</b>, <b>Optimizers</b> - rebuilt from scratch under the new convention.</p>
